# 📱 AI-Powered Rental Payment System

## 🚀 Overview
A cross-platform mobile application built with **Flutter** and **Firebase** to modernize rent payment management for landlords and tenants.

This project replaces manual, paper-based rent collection systems with a **digital, automated, and transparent platform**, reducing late payments, administrative overhead, and communication gaps.

---

## ❗ Problem Statement
In many urban rental communities (specifically Kyindi Zone, Kampala), rent management is inefficient due to:

- Manual record-keeping (paper receipts, notebooks)  
- Frequent late payments caused by lack of reminders  
- Limited payment options (cash-heavy systems)  
- Poor communication between landlords and tenants  
- Errors and disputes due to lack of real-time tracking  

These issues create financial strain for landlords and frustration for tenants.

---

## 💡 Solution
This application introduces a **mobile-first rental management system** that enables:

- Automated rent reminders  
- Secure digital payments  
- Real-time payment tracking  
- Instant digital receipts  
- Built-in communication between tenants and landlords  

---

## 🛠️ Tech Stack

### Frontend
- Flutter (Dart)

### Backend & Infrastructure
- Firebase Authentication  
- Firebase Firestore (Real-time Database)  
- Firebase Cloud Messaging (Push Notifications)  
- Firebase Hosting  

### Payments
- Flutterwave (Sandbox Integration)  
- Mobile Money (MTN, Airtel)  
- Bank Transfers  

---

## 🔑 Key Features

### 👤 Authentication & Role Management
- Secure login via Firebase Auth  
- Role-based access (Tenant vs Landlord)  

### 💸 Rent Payment System
- Real-time rent tracking (due, paid, outstanding)  
- Payment history dashboard  
- Digital receipts  

### 🔔 Automated Notifications
- Rent due reminders  
- Payment confirmations  
- Push notifications via Firebase Cloud Messaging  

### 💬 In-App Messaging
- Real-time chat between landlords and tenants  
- Reduces need for calls and physical visits  

### 💳 Payment Integration
- Flutterwave sandbox integration  
- Supports:
  - Mobile Money (MTN, Airtel)  
  - Bank transfers
